# Images Directory

Đặt file hình nền `background.png` vào thư mục này.

File hình nền sẽ được sử dụng làm background cho ứng dụng Destini.

## Yêu cầu

- Tên file: `background.png`
- Định dạng: PNG
- Kích thước khuyến nghị: 1080x1920 (tỷ lệ 9:16) hoặc lớn hơn để phù hợp với các thiết bị khác nhau
