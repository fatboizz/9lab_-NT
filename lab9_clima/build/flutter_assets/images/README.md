# Images Directory

Đặt các file hình nền vào thư mục này:

- `location_background.jpg` - Hình nền cho màn hình hiển thị thời tiết (bắt buộc)
- `city_background.jpg` - Hình nền cho màn hình nhập tên thành phố (tùy chọn)

## Lưu ý

Nếu chưa có hình nền, ứng dụng vẫn có thể chạy nhưng sẽ hiển thị lỗi khi tải hình ảnh. Bạn có thể tạm thời comment phần DecorationImage trong code nếu chưa có hình nền.
