# Lab 4: Magic 8 Ball

Dự án Magic 8 Ball là một ứng dụng Flutter cơ bản mô phỏng quả cầu tiên tri Magic 8 Ball. Đây là một bài tập giúp người mới học làm quen với quy trình phát triển ứng dụng di động bằng Flutter.

## Nội dung tìm hiểu

- Cách sử dụng Flutter SDK để xây dựng ứng dụng
- Tích hợp Material Design
- Quản lý tài nguyên (hình ảnh) trong ứng dụng Flutter
- Sử dụng StatefulWidget và quản lý trạng thái
- Cấu trúc file và thiết lập dự án Flutter qua pubspec.yaml

## Chức năng thực hiện

- Hiển thị giao diện quả cầu Magic 8 Ball với hình ảnh và câu trả lời ngẫu nhiên
- Sử dụng tài nguyên từ thư mục images/ để hiển thị hình ảnh quả cầu
- Tích hợp các widget cơ bản như AppBar, Column, Image.asset, và ElevatedButton
- Xử lý logic ngẫu nhiên để tạo câu trả lời từ danh sách có sẵn

## Chạy ứng dụng

```bash
flutter pub get
flutter run
```
