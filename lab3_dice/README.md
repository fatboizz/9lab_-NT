# Lab 3: Dice

Dự án Dice là một ứng dụng Flutter cơ bản mô phỏng việc lắc xúc xắc. Đây là một bài tập giúp người mới học làm quen với quy trình phát triển ứng dụng di động bằng Flutter.

## Nội dung tìm hiểu

- Cách sử dụng Flutter SDK để xây dựng ứng dụng
- Tích hợp Material Design
- Quản lý tài nguyên (hình ảnh) trong ứng dụng Flutter
- Sử dụng StatefulWidget và quản lý trạng thái
- Cấu trúc file và thiết lập dự án Flutter qua pubspec.yaml

## Chức năng thực hiện

- Hiển thị giao diện mô phỏng lắc xúc xắc với hình ảnh xúc xắc và nút bấm để lắc xúc xắc
- Sử dụng tài nguyên từ thư mục images/ để hiển thị hình ảnh xúc xắc
- Tích hợp các widget cơ bản như AppBar, Column, Image.asset, và ElevatedButton
- Xử lý logic ngẫu nhiên sử dụng thư viện dart:math để tạo số ngẫu nhiên mô phỏng xúc xắc

## Chạy ứng dụng

```bash
flutter pub get
flutter run
```
