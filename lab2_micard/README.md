# Lab 2: MiCard

Ứng dụng Flutter hiển thị danh thiếp kỹ thuật số với thông tin cá nhân.

## Chạy ứng dụng

```bash
flutter pub get
flutter run
```

## Tùy chỉnh

- Thay đổi tên, số điện thoại, email trong `lib/main.dart`
- Thêm ảnh đại diện vào thư mục `images/` với tên `profile.jpg`
- Kích thước khuyến nghị: 200x200px hoặc lớn hơn (hình vuông) để hiển thị tốt trong CircleAvatar
- Sau khi thêm hình ảnh, chạy `flutter pub get` để cập nhật assets
