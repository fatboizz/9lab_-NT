# Lab 5: Quizzler (Xylophone)

Ứng dụng Flutter mô phỏng đàn xylophone với 7 phím, mỗi phím phát ra một âm thanh khác nhau khi được nhấn.

## Nội dung tìm hiểu

- Cách sử dụng Flutter SDK để xây dựng ứng dụng
- Tích hợp Material Design
- Sử dụng StatefulWidget và quản lý trạng thái
- Sử dụng package audioplayers để phát âm thanh
- Quản lý tài nguyên (âm thanh) trong ứng dụng Flutter

## Chức năng thực hiện

- Hiển thị 7 phím đàn với màu sắc khác nhau
- Phát âm thanh tương ứng khi nhấn vào mỗi phím
- Sử dụng Expanded widget để phân bố không gian đều cho các phím
- Tích hợp package audioplayers để xử lý phát âm thanh

## Chạy ứng dụng

```bash
flutter pub get
flutter run
```

## Tài nguyên

Thêm các file âm thanh vào thư mục `assets/sounds/`:
- note1.wav
- note2.wav
- note3.wav
- note4.wav
- note5.wav
